# Task 模板

交付周期：0.5~1 天（完成任务项）

Task 是完成需求的具体的工作项。

**示例：**
- 设计 xxx 文档
- 开发 A 模块
- 开发 B 模块
- 测试任务

Task 的颗粒度（工作量）建议不超过一天。

```
User Story ──► Tasks
```
