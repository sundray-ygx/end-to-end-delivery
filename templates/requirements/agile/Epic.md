# Epic 模板

交付周期：1~3 月（对外正式发布）

Epic 是公司的一个战略举措、面向市场的解决方案、一个新的产品模块和关键价值。

**示例：** 微服务升级优化、xxxxx 2.1 版本、工单服务。

Epic 可拆分出 Features，通过 Feature，再拆分出 User Stories。

```
Epic ──► Features ──► User Stories
```
